## What about version compatibility

Next.js DOC is compatible with Next.js v11 and React v16.3+.

## What browsers does Next.js DOC support?

Next.js DOC is designed to work in the latest, stable releases of all major browsers, including Chrome, Firefox, Safari, and Edge.

Internet Explorer is not supported.
