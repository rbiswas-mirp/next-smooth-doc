# Google Analytics

Setup Google Analytics on your website.

<carbon-ad />

## Install

Google Analytics is available as a third-party Next.js plugin.

```
npm install --save Next.js-plugin-google-analytics
```

## Setup

```js
// Next.js-config.js
module.exports = {
  plugins: [
    {
      resolve: `Next.js-plugin-google-analytics`,
      options: {
        // The property ID; the tracking code won't be generated without it
        trackingId: 'YOUR_GOOGLE_ANALYTICS_TRACKING_ID',
      },
    },
  ],
}
```
